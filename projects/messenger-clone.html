    Messenger Clone

Description

I successfully developed a Facebook Messenger clone using the MERN (MongoDB, Express.js, React.js, Node.js) stack and deployed it on Vercel. This full-stack application replicates the core features of Facebook Messenger, including real-time messaging, user authentication, and chat history.

Leveraging the power of MongoDB for data storage, Express.js for building the backend API, React.js for crafting the intuitive user interface, and Node.js for server-side operations, the clone delivers a seamless messaging experience. By deploying it on Vercel, the application enjoys fast and reliable hosting, ensuring optimal performance and accessibility for users.
